# GenLayer Tools

A collection of reusable libraries and infrastructure tools for GenLayer Intelligent Contracts. Built as part of the GenLayer Builder Program under the Tools & Infrastructure bounty category.

All contracts were developed and tested on GenLayer Studio. No API keys required for the price feed, weather and social media libraries.

---

## What's Inside

### 1. Price Feed Oracle
**Folder:** `price-feed/`

A multi-asset price oracle covering crypto, forex and commodities. Fetches live market data from Binance and Frankfurter APIs with no API key required.

- 10 crypto pairs — BTC, ETH, SOL, BNB, XRP and more
- 7 forex pairs — USD/EUR, USD/NGN, USD/GBP and more
- 4 commodities — Gold, Silver, Platinum, Palladium
- 21 methods including batch fetch, portfolio calculator, liquidity check, AI market summary and staleness report

---

### 2. Weather Oracle
**Folder:** `weather-library/`

Live weather data for 13 cities worldwide using the Open-Meteo API. No API key required.

- Current conditions — temperature, humidity, wind speed, rainfall
- Multi-day forecast up to 7 days
- AI-powered danger alerts and city comparison
- Global weather briefing generated by the LLM

---

### 3. Social Media Oracle
**Folder:** `social-media/`

Live Hacker News data on-chain using the Firebase public API. No API key required.

- Top, new and best stories — 10 at a time
- AI-generated one-sentence news summary
- Keyword search across cached stories
- Most commented story, staleness check and full story history

---

### 4. GenLayer Key Vault
**Folder:** `genlayer_key_vault/`

A reusable security pattern for Intelligent Contracts that need to use private API keys without exposing them on-chain. Tested with a real private API key from MEXC Exchange.

- API key stored privately — never returned by any read method
- Owner access control with ownership transfer
- Rate limiting, emergency pause and key rotation
- Full audit log of every action

---

### 5. Studio UX Improvements
**Folder:** `genlayer-studio-ux/`

A document covering real UX issues and platform limitations discovered during development on GenLayer Studio. Includes 8 UX improvement suggestions and 4 known testnet limitations with workarounds.
